Javascript solutions

Picturefill, by @scottjehl

Mimics the proposed Picture element

<span data-picture data-alt="The Bayon temple in Cambodia">
  <span data-src="small.jpg"></span>
  <span data-src="medium.jpg"     data-media="(min-width: 400px)"></span>
  <span data-src="large.jpg"      data-media="(min-width: 800px)"></span>
  <span data-src="extralarge.jpg" data-media="(min-width: 1000px)"></span>
  
  <!-- Fallback content for non-JS browsers. 
       Same img src as the initial, unqualified source element. -->
  <noscript>
    <img src="small.jpg" alt="The Bayon temple in Cambodia">
  </noscript>
</span>

How it actually runs

<span data-picture="" data-alt="The Bayon temple in Cambodia">
  <span data-src="small.jpg"></span>
  <span data-src="medium.jpg" data-media="(min-width: 400px)"></span>
  <span data-src="large.jpg" data-media="(min-width: 800px)"></span>
  <span data-src="extralarge.jpg" data-media="(min-width: 1000px)">
    <img alt="The Bayon temple in Cambodia" src="extralarge.jpg">
  </span>
  
  <!-- Fallback content for non-JS browsers.
       Same img src as the initial, unqualified source element. -->
  <noscript>
    <img src="small.jpg" alt="The Bayon temple in Cambodia">
  </noscript>
</span>

Metaquery, by @benschwarz

Define your breakpoints in <meta> tags.

<meta name="breakpoint" content="phone" media="(max-width: 479px)">
<meta name="breakpoint" content="small-tablet" media="(min-width: 480px) and (max-width: 599px)">
<meta name="breakpoint" content="tablet" media="(min-width: 600px) and (max-width: 1023px)">
<meta name="breakpoint" content="widescreen" media="(min-width: 1024px)">

Use those breakpoints in two ways

First, style off those breakpoints

 .breakpoint-phone { background: red; }
 .breakpoint-small-tablet { background: green; }
 .breakpoint-tablet { background: blue; }
 .breakpoint-widescreen { background: yellow; }

And use in image markup

<img src="./images/phone.jpg" data-mq-src="./images/[breakpoint].jpg" alt="Description" >

Or

<img data-mq-src="./images/[breakpoint].jpg" alt="Description">
<noscript>
  <img src="./images/phone.jpg" alt="Description">
</noscript>

Image markup may get removed and become a plugin later

Imager, by @ResponsiveNews

This placeholder div

<div style="width: 240px">
  <div class="delayed-image-load" data-src="http://placehold.it/{width}"
                                  data-alt="alternative text"></div>
</div>

<script>
  new Imager({ availableWidths: [200, 260, 320, 600] });
</script>

Becomes this

<div style="width: 240px">
  <img src="http://placehold.it/260" data-src="http://placehold.it/{width}"
       alt="alternative text" class="image-replace">
</div>

<script>
  new Imager({ availableWidths: [200, 260, 320, 600] });
</script>
